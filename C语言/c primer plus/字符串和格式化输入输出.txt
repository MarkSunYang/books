2.字符串简介 
character string 是一个或多个字符的序列 "My name is malik, I like beatuy"，双引号不是字符串的一部分，
c语言中没有专门存储字符串变量的类型，字符串都在char类型的数组中，数组由连的存储单元组成
每个数组末尾的字符 \0  这个是空字符  null charater, c语言用它标价字符串的结束
空字符不是数字0，他是非打印字符，
这以为着数组的容量必须比待存储字符串的字符多一位。

3。常量和c 预处理
为什么使用符号常量更好？
1.常量名比数字表达的信息更多
2.多处使用的变量，统一定义
#define NAME value

const 限定符
const int MONTHS=12 //MONTHS 在程序中不可修改

符号常量和const 的区别
1.const定义的常量有类型，#define定义的没有类型
编译时可以对前者进行类型安全检查，而后者仅仅只是简单的替换。
2.const定义的常量在编译时分配内存，
const定义的常量在编译时分配内存，
3.作用域不同，const定义的作用域为该变量的作用域范围。而#define定义的常量
作用域为它的点到程序结束，当然也可以在某个地方用#undef取消。

1.如果使用scanf读取基本变量类型的值，在变量名前加一个&
