Collection 
1.1.Set 中不能包含重复的元素
1.2.List 是一个有序的集合，可以包含重复的元素，提供了按索引访问的方式

1.3.Map 是java.util中的另一个接口，他和Collection没有关系，是相互独立的，但是都是集合类的一部分
Map包含了key-value对，Map不能包含重复的 key，但是可以包含相同的value

2.集中重要的类和接口
2.1.List 有序、可重复
List中存放的对象是有序的，同事也是可以重复的，List关注的是索引，拥有一系列索引相关的方法，查询速度快。
因为往list集合插入或删除数据时，会伴随后面数据的移动，速度会变慢
2.2.Set无需、不能重复
Set中存放的对象是无序不能重复的，集合中的对象不按特定的方式排序，只是简单的把对象加入到集合中
2.3.Map(键值对、键唯一、值不唯一)

ArrayList和LinkList
ArrayList和LinkedList在用法上没有区别，但是在功能上还是有区别的。
LinkedList经常用在增删操作较多而查询操作很少的情况下，ArrayList则相反。
