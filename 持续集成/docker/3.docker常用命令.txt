docker ps 查看在运行的容器
docker ps -a 查看容器的状态

docker start/stop id/name 启动/停止某个容器
docker exec -it id 进入容器

 docker rm id/name 删除某个容器
 docker rmi id/name 删除某个镜像

 docker images 查看本地镜像


docker run 
-d: 后台运行容器，并返回容器ID；
-i: 以交互模式运行容器，通常与 -t 同时使用；
-p: 端口映射，格式为：主机(宿主)端口:容器端口
-t: 为容器重新分配一个伪输入终端，通常与 -i 同时使用；


docker rm $(sudo docker ps -a -q)   //批量删除停止的容器
ps -q :静默模式，只显示容器编号。



停用所有 container
docker stop $(docker ps -q)

删除所有container
docker rm $(docker ps -aq)

删除镜像
docker images | grep 镜像id | awk '{print $1 ":" $2}' | xargs docker rmi
